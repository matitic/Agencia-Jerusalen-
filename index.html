01 / Problema

El reto que resolvemos

La Empresa Jerusalén operaba con procesos manuales que generaban pérdidas de eficiencia y clientes.

📋

Cotización Manual

Presupuestos calculados a mano en 15–20 minutos con precios desactualizados que no reflejaban la volatilidad del mercado.

→ Ahora: segundos
📦

Inventario Descontrolado

Hojas de cálculo no sincronizadas causaban inexactitud en el stock y desabastecimiento de materiales de alta rotación.

→ Ahora: tiempo real
📉

Sin Datos Históricos

Ausencia de registros estructurados impedía identificar tendencias de precios y optimizar márgenes frente a la inflación.

→ Ahora: ML predictivo
🌐

Sin Plataforma Digital

La empresa carecía de presencia web transaccional, limitando su escalabilidad y competitividad en el mercado regional.

→ Ahora: web 24/7
Pregunta Esencial (ABR)
"¿Cómo puede un sistema web inteligente, fundamentado en Machine Learning, automatizar el proceso de cotización de materiales de construcción para reducir errores, mejorar la experiencia del cliente y fortalecer la competitividad de la Empresa Jerusalén?"
02 / Módulos

4 Módulos del Sistema

Cada módulo cubre un área crítica del negocio con tecnología de punta.

MIA
🧠

Inteligencia Analítica

  • Preprocesamiento de datos históricos
  • Modelo de regresión con Scikit-Learn
  • Predicción de precios de materiales
  • Similitud de coseno para recomendaciones
  • Visualización de tendencias
Python · Scikit-Learn
MIG
📦

Gestión de Inventario

  • Registro de SKUs y materiales
  • Control de mermas y stock crítico
  • Carga masiva vía CSV/Excel
  • Alertas automáticas de reabastecimiento
  • Gestión de proveedores
PostgreSQL · PHP
MCC
📄

Cotización Dinámica

  • Carrito de selección de materiales
  • Cálculo con tasas impositivas (IVA, IT)
  • Exportación de cotizaciones en PDF
  • Envío vía API de WhatsApp Web
  • Historial de presupuestos
ReportLab · WhatsApp API
MASS
🔐

Administración y Seguridad

  • Control de acceso por roles (RBAC)
  • Encriptación bcrypt de credenciales
  • Registro de logs de auditoría
  • Gestión de sesiones con tokens
  • Respaldos semanales automatizados
JWT · bcrypt · PostgreSQL
